ECMAScript (European Computer Manufacturers Association Script):
=========================================================
1.ECMAScript is a scripting-language specification standardized by 
   Ecma International in ECMA-262 and ISO/IEC 16262. 

2. It was created to standardize JavaScript, so as to foster multiple independent implementations.

3.Designed by Brendan Eich, Ecma International.

4.Influenced languages: 
	Self,
	HyperTalk,
	AWK,
	 C,
	CoffeeScript
	 Perl, 
	Python, 
	Java, 
	Scheme.

5.	Developed by	Sun Microsystems,Ecma International
	Initial release	June 1997; 22 years ago

7.ECMAScript is often abbreviated to ES.

8.JavaScript Vs ECMAScript:
========================
JavaScript was developed for Netscape. 
Netscape 2 was the first browser to run JavaScript.
After Netscape the Mozilla foundation continued to develop JavaScript for the Firefox browser.
The latest JavaScript version was 1.8.5. (Identical to ECMAScript 5).
ECMAScript was developed by ECMA International after the organization adopted JavaScript.
The first edition of ECMAScript was released in 1997.

Microsoft TypeScript:
------------------------------
1.Microsoft TypeScript is an open-source programming language developed 
and maintained by Microsoft. 

2.It is a strict syntactical superset of JavaScript, and adds optional static typing to the language. 

3.TypeScript is designed for development of large applications and transcompiles to JavaScript. 

4.Key points:

Stable release		: 3.4.5 / 24 April 2019
Filename extensions	: ts,.tsx
First appeared		: 1 October 2012
Designed by		: Microsoft Corporation
Officil Site		:https://www.typescriptlang.org/
